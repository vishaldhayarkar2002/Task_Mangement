## Task Management

 Description: A system for creating, assigning, and tracking tasks across teams.
 Functional Requirements:
 Task Creation:
 Create tasks with attributes: title, description, priority, due date, and tags.
 Assign tasks to users or teams.
 Task Tracking:
 Status updates (Open, In Progress, Completed).
 Add comments or attachments to tasks.
 Dashboard:
 View assigned tasks in a list, kanban, or calendar view.
 Filter tasks by status, due date, or priority.
 Notifications:
 Notify users of task assignments or due dates.
 Reports:
 View task completion stats for a user or team.
 Track overdue tasks.
 User Permissions:
 Only allow task creators or admins to edit/delete tasks
